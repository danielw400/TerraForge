# assets/

Estrutura de recursos utilizada pelo projeto TerraForge.

Subpastas:
- models/ - modelos 3D
- textures/ - texturas e sprites
- audio/ - efeitos sonoros e músicas
- shaders/ - shaders e materiais
- ui/ - elementos gráficos de interface

Adicione arquivos binários (modelos, texturas, áudio) aqui conforme necessário.
